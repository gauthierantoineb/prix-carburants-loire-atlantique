# Dashboard Looker Studio

## Objectif
Aider l’utilisateur à identifier les stations les moins chères dans un rayon donné autour de Nantes centre.

## Filtres
- carburant
- distance_km
- prix

## KPI
- nombre de stations
- prix moyen
- prix minimum

## Visualisations
- distribution des prix
- carte des stations
- tableau des stations trié par prix

## Lecture métier
Le dashboard permet de répondre rapidement à la question :
où faire le plein au meilleur prix dans le périmètre sélectionné ?
